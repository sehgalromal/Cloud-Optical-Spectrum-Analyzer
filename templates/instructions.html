Follow these sequence of actions for retrieving the OSA trace and display the results on graph based on these commands

  1. Click on the start button to initiate aquisition
  2. Click on the "Fetch Single" button to fetch single trace
  3. If you want to set x-axis limits; specify the minimum and maximum value (b/w 1515 and 1580) by executing the "/cmd/LIM/[min,max]" command as present on main page itself
  4. Click on the "Display Graph" button to generate graph
Click on the stop button to stop the aquisition

You can execute any command in different combinations and see the results

Few instructions while interacting with Graph and Web Application

  • Select the Specific by mouse of Graph you want to zoom in
  • For moving the view of the graph when Panning; Select the specific graph first and then scroll left or right.
  • For downloading or viewing the graph in full screen mode, click on top-right menu bar for available options
  • If any issue occurs while interacting with the application, please reload it
