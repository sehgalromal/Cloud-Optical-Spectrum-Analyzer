Follow these sequence of actions for retrieving the OSA trace and display the results on graph based on these commands

  1. Click on the start button to initiate aquisition
  2. Click on the "Fetch Single" button to fetch single trace
  3. If you want to set x-axis limits; specify the minimum and maximum value (b/w 1515 and 1580) by executing the "/cmd/LIM/[min,max]" command as present on main page itself
  4. Click on the "Display Graph" button to generate graph
Click on the stop button to stop the aquisition

You can execute any command in different combinations and see the desired results

Few instructions while interacting with Graph and Web Application

  • Select the Specific by mouse of Graph you want to zoom in
  • When Panning the graph; Select the specific graph first and then scroll left or right horizontally for viewing specific area
  • For downloading or viewing the graph in full screen mode, click on top-right menu bar for available options
  • If any issue occurs while interacting with the application, please reload it

For accessing different api via endpoints: Refer to the following table

Command Rest API Endpoint
/cmd https://cloud-osa-optical-analyzer.herokuapp.com + /cmd returns command prompt
/cmd/IDN https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/IDN returns x-axis limits in m
/cmd/LIM https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/LIM returns device identification string
/cmd/LIM/[min max] Specify min and max limits when accssing the url: https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/LIM/ + [min,max] sets x-axis limits in nm
/cmd/ECHO/string Specify String when accessing this url: https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/ECHO/ + string Emulates query command and sends a string to API, will get the same string back
/cmd/PING https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/PING Returns PONG
/cmd/START https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/START sets instrument state to continues acquisition
/cmd/STOP https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/STOP sets instrument state to IDLE
/cmd/SINGLE https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/SINGLE starts a single scan (blocking operation, single scan takes few seconds)
/cmd/STATE https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/STATE returns instrument state
/cmd/TRACE https://cloud-osa-optical-analyzer.herokuapp.com + /cmd/TRACE returns OSA trace in json format
